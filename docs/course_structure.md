# Course/Project structure

Let's create a directory structure in order to organize the data as we produce it:

```{bash}
# ~ is a shortcut for the home directory
cd ~
# create the directories that we need for the course
mkdir rnaseq_course
mkdir rnaseq_course/singularity_image
mkdir rnaseq_course/raw_data
mkdir rnaseq_course/quality_control
mkdir rnaseq_course/trimming
mkdir rnaseq_course/mapping
mkdir rnaseq_course/differential_expression
```
